# GitHub Review Commands

Read only the command set needed for the current review task:

- `gh-review-request.md` - request AI review and poll PR state.
- `gh-review-list.md` - list top-level and inline review comments.
- `gh-review-reply.md` - reply to inline review comments and update PR
  comments.
- `gh-review-threads.md` - query review thread resolution state with GraphQL.
